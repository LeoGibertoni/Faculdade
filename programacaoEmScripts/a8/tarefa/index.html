O que é AJAX?

AJAX (Asynchronous JavaScript and XML) é uma técnica de desenvolvimento web que permite atualizar partes de uma página sem precisar recarregá-la por completo. Isso proporciona uma experiência mais fluida e interativa para o usuário.

AJAX funciona através de requisições assíncronas ao servidor, utilizando objetos como XMLHttpRequest ou, mais recentemente, a API fetch().

Exemplo com fetch()

<script>
  fetch('https://jsonplaceholder.typicode.com/posts/1')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error('Erro:', error));
</script>

No exemplo acima, usamos fetch() para buscar dados de um servidor de testes e mostramos o resultado no console do navegador. Isso acontece sem que a página precise ser recarregada.

AJAX é amplamente utilizado em sites modernos, como redes sociais, aplicativos de mensagens, formulários dinâmicos e muito mais.
